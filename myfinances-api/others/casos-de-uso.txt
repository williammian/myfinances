- Efetuar Login

Caso o usuário esteja cadastrado no sistema, poderá efetuar o login, dessa forma o sistema exigirá que o mesmo
informe obrigatoriamente o email e senha cadastrados. Caso não esteja cadastrado, o sistema deverá disponibilizar um
formulário para cadastro do usuário.

- Cadastro de Usuários

Sistema deverá permitir que qualquer usuário possa efetuar o seu cadastro para acesso ao sistema.
Os dados do usuário que deverão ser exigidos pelo sistema são: nome, email, e senha.
O email deverá ser único, a fim de evitar cadastros duplos.

- Cadastro de Lançamentos

Deseja-se cadastrar os lançamentos de Receita e Despesa de cada mês, com seu valor e descrição.
Ao cadastrar, ele ficará pendente. Haverá opção de cancelá-lo ou efetivá-lo.
Ao efetivá-lo, ele incorpora o saldo. Os usuários poderão consultar seus lançamentos, 
filtrando-os por ano, mês, descrição ou tipo de lançamento.
